Download MinGW

First, you need to download MinGW, which is a minimalist development environment for native Microsoft Windows applications.

https://sourceforge.net/projects/mingw/

Select the GCC g++ Package and Base

Select GCC g++ package

Select base package

During the installation process, make sure to select the GCC g++ package and the base package. These packages are necessary for compiling C programs.

Install and Apply Changes

Click on the installation button and apply changes to complete the setup. This will install the necessary tools on your system.

Set Environment Variables

Copy the bin folder path

Set the path in environment variables

Go to the installation folder of MinGW, select its bin folder and copy its path. Then, type "environment variables" in the Windows search bar, click on Path within user variables or system variables, and add the copied path.

Install Code Runner Extension

In Visual Studio Code, install the Code Runner extension. This extension allows you to run your C code directly from the editor. After installation, restart Visual Studio Code to apply the changes.

Basic C Program Format

#include <stdio.h>

int main() {

printf("Hello, World!");

return 0;

}

This is a basic C program that prints "Hello, World!" to the console. The #include <stdio.h> directive includes the Standard Input Output library, which allows you to use the printf function.
